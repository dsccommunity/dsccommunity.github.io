Community Calls

The DSC Community calls are schedule and you can add them to your calender using the ICS file (the ICS is also available to download as a ZIP file).

How to Join

The calls are open for everyone in the community.

Microsoft Teams

Join through the Microsoft Teams Meeting

Phone

+1 323-849-4874 United States, Los Angeles (Toll), or Local numbers.

Conference ID: 503 431 76#

Participants using the Join by Phone option must dial the full number for call access.

Community Call Agenda

To the left you will find the agenda for the next community call and the notes for the previous community calls. Please feel free to send in PR’s to add items you feel need to be discussed in the next community call in the agenda.

The file to update for new agenda items is content/community_calls/next_call.en.md

Add next community call agenda and update notes to the web site

Make sure after the call that the agenda (next_call.en.md) is renamed appropriately, and the weight: 99x decremented from the last notes.

Then run the command below to create a new file under content/community_calls.

hugo new community_calls/next_call.en.md
